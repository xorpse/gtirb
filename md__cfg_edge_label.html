CfgEdgeLabel

A CfgEdgeLabel represents the optional label on an edge (CfgEdge) in the interprocedural control flowgraph" (CFG).

The corresponding Protobuf message type is EdgeLabel.

Guaranteed Properties

  • Each CfgEdgeLabel object must have the following information, and the API must provide functionality for getting and setting each.
    • conditional
    • direct
    • type
